## Installation

See the [README](https://github.com/STARTcloud/patchpanel/blob/main/README.md)
for installation steps — `apt install patchpanel haproxy` on Debian once
the STARTcloud apt repo is configured, or install the HAProxy add-on in
Home Assistant. Detailed guides will land at the
[Architecture](architecture/) and [API Reference](api/) pages on this
site.
